== Description ==

This plugin will add Social Profile Fields into the user detail page.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'CS Add Social Fields to Admin Area'
3. Activate CS Add Social Fields to Admin Area from your Plugins page.

= From WordPress.org =

1. Download CS Add Social Fields to Admin Area
2. Upload the 'cs-add-social-fields-to-admin-area' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate CS Add Social Fields to Admin Area from your Plugins page.
